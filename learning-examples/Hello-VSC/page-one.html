Lists

This is how you create an un-ordered list:

  • Add a "ul" element (ul = unordered list li = list item)
  • Add each item in its own "li" element
  • They don't need to be in any particular order

  • HTML has strict rules about what elements can go inside other elements. If you are ever unsure do a quick google search for "MDN "< some-element >"". This line is a paragraph element nested in a list element

This is what an ordered list looks like (it is a numbered list)

  1. Notice the new "ol" (ordered list) element wrapping everything
  2. But the list item elements are the same (li)
  3. Also note how the numbers increment on there own
  4. You should be noticing the list is in a precise order, because after all this is an ordered list

Inline Elements

Sometimes, you need to draw attention to a particular word or phrase, This can be doe with an inline element like "< em>< /em >"

Other times you need to Strongly emphasize the importance of a word or phrase. To do this use the "< strong >" element

And sometimes you need to shout!

this can be done by nesting a < strong > element in an < em > element or visa-versa
